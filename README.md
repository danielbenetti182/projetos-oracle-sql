# Portfólio de Modelagem e Consulta de Dados

Portfólio de modelagem de dados e desenvolvimento de consultas estruturadas
utilizando **Oracle SQL** e **Oracle APEX**, desenvolvido durante um curso extra
de Banco de Dados Oracle.

Este repositório reúne três projetos com contextos de negócio diferentes,
demonstrando tanto a capacidade de **modelar bancos de dados do zero** quanto de
**interpretar e consultar uma arquitetura de dados já existente**.

## 🛠️ Tecnologias Utilizadas

- **Oracle SQL**
- **Oracle APEX**
- **Modelagem ER** (Entidade-Relacionamento / Diagrama Lógico)

## 📁 Estrutura do Repositório

### [`/Gestao_Lanchonete`](./Gestao_Lanchonete)
Banco de dados de uma lanchonete, com modelo de dados pré-fornecido pelo
professor. O destaque deste projeto é a habilidade de ler, compreender e
manipular uma arquitetura existente, escrevendo consultas complexas com
`JOINs`, subqueries e agregações de faturamento (mensal, por categoria e por
forma de pagamento).

### [`/Sistema_Cursos_Extracurriculares`](./Sistema_Cursos_Extracurriculares)
Modelagem conceitual e lógica, feita do zero, de um sistema de gestão de
cursos extracurriculares — cobrindo Alunos, Matrículas, Cursos, Professores e
Controle de Presença.

### [`/Controle_Biblioteca`](./Controle_Biblioteca)
Modelagem entidade-relacionamento de um sistema de controle de biblioteca,
com foco no fluxo de Materiais, Autores, Usuários, Funcionários e Empréstimos.

## 🎯 Sobre este portfólio

Este material foi produzido durante um curso extracurricular de Oracle SQL,
como preparação técnica para atuação em ambientes que utilizam esse banco de
dados no dia a dia.
